## Install ##

Using [Homebrew](http://brew.sh/) (OS X / Linux)

```shell
brew install kulapard/tap/gol
```

## Update ##

Using [Homebrew](http://brew.sh/) (OS X / Linux)

```shell
brew upgrade kulapard/tap/gol
```
